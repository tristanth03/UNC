1. Introduction

Compound interest is the engine behind long-term wealth accumulation. The future value of an investment with regular contributions is given by:

\[FV = P\,(1+r)^{n} \;+\; C\,\frac{(1+r)^{n}-1}{r}\]

where \(P\) is the initial principal, \(C\) is the periodic contribution, \(r\) is the periodic interest rate, and \(n\) is the number of periods.

This notebook provides an interactive Plotly dashboard to explore how changes in annual return rate, monthly contribution, and investment horizon affect portfolio growth — and how much of the final balance is pure interest versus contributions.

2. Interactive Savings Growth Chart

Use the three sliders to adjust: - Annual return (%) — expected average annual rate of return - Monthly contribution ($) — how much you add each month - Years — investment time horizon

The shaded area shows cumulative contributions; the line above shows total portfolio value (contributions + compounded interest).

import json
from pathlib import Path
import numpy as np
import plotly.graph_objects as go

# ── Load theme ───────────────────────────────────────────────────
THEME_NAME = "navy_vault"

def _find_theme_file() -> Path:
    try:
        anchor = Path(__file__).resolve().parent
    except NameError:
        anchor = Path.cwd()
    for parent in [anchor, *anchor.parents]:
        candidate = parent / "theme" / "font_mapping.json"
        if candidate.is_file():
            return candidate
    raise FileNotFoundError("Could not find theme/font_mapping.json")

_cfg = json.loads(_find_theme_file().read_text())
C    = _cfg["themes"][THEME_NAME]
FONT = _cfg["font"]["family"]
FSIZ = _cfg["font"]["size"]

# ── Parameters ───────────────────────────────────────────────────
P0 = 10_000          # initial deposit
default_rate = 7.0   # %
default_monthly = 500
default_years = 30

months = np.arange(0, default_years * 12 + 1)

def portfolio(P0, annual_rate, monthly, n_months):
    r = annual_rate / 100 / 12
    bal = np.zeros(n_months + 1)
    contrib = np.zeros(n_months + 1)
    bal[0] = P0; contrib[0] = P0
    for i in range(1, n_months + 1):
        bal[i] = bal[i-1] * (1 + r) + monthly
        contrib[i] = contrib[i-1] + monthly
    return bal, contrib

bal, contrib = portfolio(P0, default_rate, default_monthly, len(months)-1)
years_ax = months / 12

fig = go.Figure()
fig.add_trace(go.Scatter(
    x=years_ax, y=bal, mode="lines", name="Total value",
    line=dict(color=C["mid"], width=2.5),
))
fig.add_trace(go.Scatter(
    x=years_ax, y=contrib, mode="lines", name="Contributions",
    line=dict(color=C["mid"], width=2), fill="tozeroy",
    fillcolor="rgba(14,96,144,0.12)",
))

# ── Sliders ──────────────────────────────────────────────────────
slider_style = dict(
    bordercolor=C["silver"], bgcolor=C["ci"],
    activebgcolor=C["mid"], tickcolor=C["silver"],
    font=dict(family=FONT, size=FSIZ["sub"], color=C["subtext"]),
    ticklen=4, minorticklen=0, len=0.92, x=0.04,
)
_cv = dict(font=dict(family=FONT, size=FSIZ["label"], color=C["text"]))

rate_steps = []
for rate in np.arange(1, 15.5, 0.5):
    b, c = portfolio(P0, rate, default_monthly, default_years * 12)
    yrs = np.arange(len(b)) / 12
    rate_steps.append(dict(method="restyle",
        args=[{"y": [b.tolist(), c.tolist()]}], label=f"{rate:.1f}%"))

contrib_steps = []
for m in range(0, 2100, 100):
    b, c = portfolio(P0, default_rate, m, default_years * 12)
    contrib_steps.append(dict(method="restyle",
        args=[{"y": [b.tolist(), c.tolist()]}], label=f"${m}"))

year_steps = []
for yr in range(5, 51, 5):
    b, c = portfolio(P0, default_rate, default_monthly, yr * 12)
    yrs = np.arange(len(b)) / 12
    year_steps.append(dict(method="restyle",
        args=[{"x": [yrs.tolist(), yrs.tolist()], "y": [b.tolist(), c.tolist()]}],
        label=f"{yr}y"))

fig.update_layout(
    sliders=[
        dict(**slider_style, active=12, currentvalue={**_cv, "prefix": "Annual return: "}, steps=rate_steps, y=-0.02, pad=dict(t=30,b=10)),
        dict(**slider_style, active=5,  currentvalue={**_cv, "prefix": "Monthly contribution: "}, steps=contrib_steps, y=-0.14, pad=dict(t=30,b=10)),
        dict(**slider_style, active=5,  currentvalue={**_cv, "prefix": "Horizon: "}, steps=year_steps, y=-0.26, pad=dict(t=30,b=10)),
    ],
    title=dict(text="Savings Growth: Contributions vs Compound Interest",
               font=dict(family=FONT, size=16, color=C["text"]), x=0.04, xanchor="left"),
    paper_bgcolor=C["canvas"], plot_bgcolor=C["canvas"],
    font=dict(family=FONT, color=C["text"]),
    width=860, height=580, margin=dict(l=70,r=30,t=55,b=185),
    legend=dict(x=0.04, y=0.96, bgcolor="rgba(0,0,0,0)",
                font=dict(size=FSIZ["label"], color=C["text"])),
    xaxis=dict(title=dict(text="Years", font=dict(size=FSIZ["label"], color=C["subtext"])),
               tickfont=dict(size=FSIZ["tick"], color=C["subtext"]),
               gridcolor=C["grid"], zerolinecolor=C["silver"], zerolinewidth=0.8),
    yaxis=dict(title=dict(text="Portfolio Value ($)", font=dict(size=FSIZ["label"], color=C["subtext"])),
               tickfont=dict(size=FSIZ["tick"], color=C["subtext"]), tickprefix="$", tickformat=",.0f",
               gridcolor=C["grid"], zerolinecolor=C["silver"], zerolinewidth=0.8),
)
fig.show()

3. Key Takeaways

Observations:

- Even modest monthly contributions compound dramatically over long horizons — the gap between the two curves is pure interest. - Doubling the time horizon far exceeds doubling the contribution, illustrating the non-linear nature of exponential growth. - At \(7\%\) annual return over 30 years, a \(\$500/month\) contribution on a \(\$10,000\) base grows to over \(\$600,000\) — of which roughly \(\$190,000\) is contributions and \(\$410,000\)+ is compound interest.

The exponential factor \((1+r)^n\) is doing most of the work; time in the market dominates timing the market.

4. Mathematical Notes

The closed-form future-value formula assumes a constant rate \(r\) per period:

\[FV = P_0(1+r)^n + C\,\frac{(1+r)^n - 1}{r}\]

For variable rates, one must iterate:

\[B_{k+1} = B_k\,(1 + r_k) + C_k\]

The Rule of 72 gives a quick estimate: money doubles in roughly \(72/r\) years at \(r\%\) annual return. At \(7\%\), that is about 10.3 years.
